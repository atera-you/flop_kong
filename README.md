これらの総数を計算して出す。その際かぶりが起きる場合は最も大きな役を適応させることに注意。

レンジ入力は最小のペアとスートとオフスートで入力する。
ペア　シンプル　入れた数以上の数字のペアをレンジに入れる
スートやオフスート　組み合わせをどう入力するべきか。A〜3それぞれのタブを作った上で相方の最低値を選ぶとそれ以上がカウントされるようにするのが一番無難かな。
